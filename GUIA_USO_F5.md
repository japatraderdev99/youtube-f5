### **PASSO 1: Preparar Lista de Vídeos**
Crie arquivo CSV com colunas:
- `titulo`: Título atual do vídeo
- `descricao`: Descrição atual
- `tags`: Tags separadas por vírgula

### **PASSO 2: Executar Otimização**
```bash

```

O sistema gera:
